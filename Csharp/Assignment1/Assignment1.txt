Q2:
A data type is a value type if it holds a data value within its own memory space. It means the variables of these data types directly contain values.
A reference type doesn't store the value directly. Instead, it store a reference to the address where value is being stored.

Boxing refers to conversion of a data-type to a reference-type value
Unboxing fefers to conversion of a reference-type to a data-type value

CLR is the basic and Virtual Machine component of the .NET Framework.

Managed resources are those that are pure . NET code and managed by the runtime and are under its direct control. 

Unmanaged resources are those that are notdirectly under the control of the garbage collector. File handles, pinned memory, COM objects, database connections etc.
